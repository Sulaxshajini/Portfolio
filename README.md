## 📖 About

This is the personal portfolio website of **Sulaxshajini Kumarakulasingam**, an IT undergraduate and aspiring software engineer. The site showcases her technical skills, academic projects, education, professional strengths, and contact information — all in a single, beautifully crafted HTML file.

The design is built around a **Glacier & Ink** colour palette with a glassmorphism aesthetic, animated ambient background orbs, scroll-reveal effects, and a fully responsive layout optimised for desktop, tablet, and mobile.

---

## ✨ Features

- 🎨 **Glassmorphism UI** — Frosted-glass cards with `backdrop-filter` blur and teal accent borders
- 🌊 **Animated ambient background** — Five layered gradient orbs that drift continuously
- 📜 **Scroll-reveal animations** — Sections fade and slide in as the user scrolls
- 🤖 **Interactive chatbot** — Portfolio assistant for visitor engagement
- 🏠 **Hero section** — Profile card with rotating halo animation, ticket-style info fields, and CTA buttons
- 💼 **Projects showcase** — Cards for the Sunnyside Dairy Management System and Portfolio Website
- 🛠️ **Skills section** — Pill-style tags across Frontend, Backend, Databases, and Soft Skills
- 🎓 **Education timeline** — Academic history with a vertical timeline component
- 🌍 **Languages section** — Progress-bar cards for Tamil, English, and Sinhala
- 📞 **Contact section** — Dark gradient section with social icon buttons and a reference card
- 📱 **Fully responsive** — Breakpoints at 900px, 800px, 640px, and 580px

## 🛠️ Tech Stack

| Layer | Detail |
|---|---|
| Markup | HTML5 (single `index.html`, 1832 lines) |
| Styling | CSS3 — custom design tokens, `backdrop-filter`, CSS Grid, Flexbox, `@keyframes` |
| Fonts | Playfair Display (display), Inter (body), JetBrains Mono (mono/labels) via Google Fonts |
| Scripting | Vanilla JavaScript — scroll-reveal observer, nav scroll class, chatbot logic |
| Hosting | GitHub Pages |

### 🎨 Design Tokens (Glacier & Ink Palette)

| Token | Value | Use |
|---|---|---|
| `--ice-400` | `#34B6C4` | Primary accent, buttons, halos |
| `--ice-600` | `#0E7480` | Dark teal, links, labels |
| `--ink-900` | `#07131D` | Dark card backgrounds |
| `--cream` | `#F7FBFC` | Page background |
| `--glass-white` | `rgba(255,255,255,0.55)` | Glass card surface |

## 🚀 Getting Started

### View locally

```bash
git clone https://github.com/Sulaxshajini/Portfolio.git
cd Portfolio
```

Then open `index.html` in any modern browser. No server, no `npm install` needed.

### With VS Code Live Server

1. Install the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer)
2. Right-click `index.html` → **Open with Live Server**

### Deploy to GitHub Pages

1. Go to **Settings → Pages** in the repository
2. Set source to `main` branch, `/ (root)`
3. Your site will be live at `https://sulaxshajini.github.io/Portfolio/`
